# Forecast Dashboard (React + Vite)

A web-based forecasting dashboard with:
- Forecast vs Actual vs Budget views
- KPI cards for Revenue, Covers, Average Spend, and Margin
- Time levels: Day, Week, Month, YTD
- Channels: Brasserie, Snackbar, Retail
- Filters for Park, Period, and Outlet
- Editable forecast inputs with instant recalculation
- Traffic light indicators (red / orange / green)
- Built-in demo dataset for immediate usage

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
